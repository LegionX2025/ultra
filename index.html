  • Completed audits are formatted and isolated into the contract_audits collection in your us-east-1 MongoDB cluster.
  • The schema includes address, chain, vulnerabilities array, risk_level, summary, and the source code snippet itself.

  • Exposed a new endpoint at [GET] /api/export_cex_sweep.
  • Implemented streaming CSV encoding directly from the contract_audits collection, avoiding massive memory bloat on large datasets.
  • The Global CEX Monitor UI now features a prominent indigo EXPORT CSV DOSSIER button in the header stripe.
